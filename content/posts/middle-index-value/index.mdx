## 문제 설명

[문제 보러가기](https://school.programmers.co.kr/learn/courses/30/lessons/120811)

중앙값은 어떤 주어진 값들을 크기의 순서대로 정렬했을 때 가장 중앙에 위치하는 값을 의미합니다.  
예를 들어 1, 2, 7, 10, 11의 중앙값은 7입니다.  
정수 배열 array가 매개변수로 주어질 때, 중앙값을 return 하도록 solution 함수를 완성해보세요.
단, array의 길이는 **홀수**입니다.

## Example

```
array = [1, 2, 7, 10, 11]
> 7

array = [9, -1, 0]
> 0
```

## 풀이

```js
function solution(array) {
  const sortArr = array.sort((a, b) => a - b);
  const middleIndex = Math.floor(array.length / 2);

  return sortArr[middleIndex];
}
```

배열이 홀수의 길이로 내려오기 때문에
배열의 길이를 2로 나누면 x.5 의 값들이 나온다.
배열은 0부터 시작하기 때문에 .5를 내리면 중앙값이 나온다!

```
sortArray = [1, 2, 7, 10, 11]
sortArray.length = 5
5 / 2 = 2.5 => Math.floor(2.5) => 2
sortArray[2] = 7
> 7

sortArray = [-1, 0, 9]
sortArray.length = 3
3 / 2 = 1.5 => Math.floor(1.5) => 1
sortArray[1] = 0
> 0
```
