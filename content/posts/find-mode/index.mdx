## 문제 설명

[문제 보러가기](https://school.programmers.co.kr/learn/courses/30/lessons/120812)

최빈값은 주어진 값 중에서 가장 자주 나오는 값을 의미합니다.  
정수 배열 array가 매개변수로 주어질 때, 최빈값을 return 하도록 solution 함수를 완성해보세요.  
최빈값이 여러 개면 `-1`을 return 합니다.

## Example

```
array = [1, 2, 3, 3, 3, 4]
> 3

array = [1, 1, 2, 2]
> -1

array = [1]
> 1
```

## 풀이

```js
function solution(array) {
  // 배열의 index는 0에서 시작하기 때문에
  // array에서 가장 큰 수만큼의 길이인 배열을 만들고 0으로 채워준다
  let hashArr = new Array(Math.max(...array) + 1).fill(0);

  // array의 el들을 index로 하는 hashArr에 해당 el를 카운팅해준다
  array.forEach((el) => hashArr[el]++);

  // hashArr에서 가장 큰 값 = array에서 가장 많이 있다는 뜻
  const hashArrMax = Math.max(...hashArr);

  // hashArr에서 가장 큰 값과 같은 값이 있는지 필터링한다
  // newArr의 길이가 1보다 크면 최빈값이 여러개라는 뜻!
  const newArr = hashArr.filter((el) => el === hashArrMax);

  // newArr의 길이가 1보다 크면 최빈값이 여러개이므로 -1 return
  // 아니면 hashArr에서 가장 큰 값을 value로 하는 index를 찾는다
  return newArr.length > 1 ? -1 : hashArr.indexOf(hashArrMax);
}
```
